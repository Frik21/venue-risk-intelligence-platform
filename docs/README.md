# VenueGuard

VenueGuard is an Operational Intelligence and Operational Planning platform built for security professionals.

## Philosophy

**Planning powered by Intelligence.**

VenueGuard reduces uncertainty by transforming intelligence into clear operational planning and decision support.

**VenueGuard informs. Security professionals decide.**

## Source of Truth

Before making architectural or product changes, always consult:

- `docs/PROJECT_CONTEXT.md`
- `docs/Product-Constitution.md`
- `docs/Operations-Centre-v1.md`

These documents define the product philosophy and architecture.

## Repository

This repository contains the VenueGuard platform, shared libraries, database schema, and supporting services.

## Development

Install dependencies:

```bash
pnpm install
```

Run type checking:

```bash
pnpm run typecheck
```

Build the project:

```bash
pnpm run build
```
